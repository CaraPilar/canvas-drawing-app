# canvas-drawing-app
HTML5 Canvas Drawing app based off of Codecourse Tutorial: https://www.youtube.com/watch?v=m4sioSqlXhQ


Features

[x] Color Swatches
[x] Eraser
[x] Background Fill Tool
[x] Brush
[x] Pen
[x] Pencil
[x] Save
[x] Clear Canvas
[x] Undo/Redo
[] Advanced Color Tool
[] Text Input
[] Drag & Drop Clip Art/Stickers
[] Develop app in React
